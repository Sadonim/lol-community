# LOL 커뮤니티

리그 오브 레전드 유저를 위한 커뮤니티 포럼입니다.

🔗 **[라이브 데모](https://lol-community-testbuild.vercel.app)**

---

## 주요 기능

- **게시판** — 자유, 공략, 팀원 모집 등 게시판별 글 작성 / 댓글 / 투표
- **Riot 계정 연동** — Riot ID로 소환사 정보(티어, 승률) 연동
- **랭킹** — 연동된 소환사들의 티어 랭킹
- **알림** — 댓글, 답글, 좋아요 실시간 알림
- **프로필 / 설정** — 닉네임 변경, 비밀번호 변경, Riot 계정 관리

---

## 기술 스택

| 분류 | 사용 기술 |
|------|-----------|
| Framework | Next.js 16 (App Router) |
| API | tRPC v11 |
| Auth | NextAuth v5 (JWT) |
| DB | PostgreSQL (Supabase) + Prisma |
| UI | Tailwind CSS + shadcn/ui |
| 배포 | Vercel |
